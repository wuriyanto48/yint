### yint (yes its network tool)

Trust me, its just a useless networking tool that uses a `C++`, `NodeJs` and `Chrome's V8`(https://v8docs.nodesource.com/)

Getting started

Install `node-gyp`
```shell
$ npm install -g node-gyp
```

Configure and Build with `node-gyp`
```shell
$ node-gyp configure
$ node-gyp build
```

Test project
```shell
$ npm start
```
