### yint (yes its network tool)

Its just a useless networking tool that bind a `C++` code into `NodeJs` that uses `Chrome's V8`(https://v8docs.nodesource.com/)

Getting started

Install `node-gyp`
```shell
$ npm install -g node-gyp
```

Configure and Build with `node-gyp`
```shell
$ node-gyp configure
$ node-gyp build
```

Test project
```shell
$ npm start
```
